Drone Link

A File Organizer Script for Drone Media Files

About

The Drone Link project is a file organizer script designed to manage and organize drone media files, ensuring a clean and organized directory structure. It utilizes Node.js and provides functionalities to copy, organize, and remove files from a source directory to a destination directory based on their extensions.

Features

  • Copy and organize files by their extensions into subdirectories.
  • Remove old files from the source directory after copying.
  • Progress tracking with a visual progress bar.
  • Easy-to-use command-line interface for specifying directories.

Usage

To use the Drone Link script:

  1. Clone or download the repository to your local machine.
  2. Navigate to the script directory using a terminal.
  3. Run the script using the command: node index.js
  4. Follow the prompts to specify source and destination directories.
  5. View the progress bar as the script copies and organizes files.
  6. Old files will be removed from the source directory after copying.

Customization

You can customize the behavior of the script by modifying the code in index.js.
